# 📄 About this Repository

This repository contains the data and code used in the paper:

**AI in science: When and where it makes a difference**

Authors: Bianchini, Di Girolamo, Ravet, Arranz (2026)

It includes:
- Dataset 
- Code to reproduce all results



# 📁 Repository Structure

- `/DATA`       → Cleaned datasets used in the paper 
- `/CODE`       → Python script for figures and regressions 


# 🔄 Reproducibility

All figures and tables can be regenerated by running the scripts in `/CODE`.

A reproducibility header is included in each script with instructions on required packages and datasets.
